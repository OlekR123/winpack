# WinPack

Веб-приложение для персонализированной установки программ и настройки Windows.
Пользователь выбирает нужные программы и системные настройки, а приложение
формирует готовый к исполнению PowerShell-скрипт. Установка программ выполняется
через менеджер пакетов Winget.

## Возможности

- Каталоги программ и системных настроек с разбивкой по категориям.
- Поиск пакетов через Winget и генерация PowerShell-скриптов.
- Регистрация с подтверждением почты по одноразовому коду.
- Сохранение персональных конфигураций пользователей.
- Административная панель со статистикой использования настроек.

## Технологии

- **Клиент:** Vue 3, Vite.
- **Сервер:** Node.js, Express.
- **База данных:** PostgreSQL (Supabase), бизнес-логика в хранимых функциях.
- **Авторизация:** JWT, хеширование паролей bcrypt.

## Установка и запуск

Требуется Node.js 20.19+.

```sh
npm install      # установка зависимостей
npm run dev:all  # запуск клиента и сервера
npm run test     # запуск тестов
```

Перед запуском создайте файл `.env` с параметрами подключения к базе данных
(PGHOST, PGPORT, PGDATABASE, PGUSER, PGPASSWORD) и приложения (JWT_SECRET,
RESEND_API_KEY, FROM_EMAIL).
